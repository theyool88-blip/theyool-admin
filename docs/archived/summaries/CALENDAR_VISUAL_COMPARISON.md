## Design System Tokens

### Border Radius
- BEFORE: `rounded-md` (6px), `rounded-lg` (8px)
- AFTER: `rounded-lg` (8px), `rounded-xl` (12px)

### Shadows
- BEFORE: `shadow` (medium shadow)
- AFTER: `shadow-sm` (subtle), `shadow-md` (on interaction)

### Borders
- BEFORE: `border` (1px all sides), `border-2` (thick all sides)
- AFTER: `border-l-4` (accent only), `border border-gray-100` (subtle outline)

### Typography
- BEFORE: 일관되지 않은 크기
- AFTER: 명확한 계층
  - `text-2xl` (제목)
  - `text-xl` (부제목)
  - `text-sm` (설명)
  - `text-xs` (본문)
  - `text-[10px]` (작은 레이블)

### Spacing
- BEFORE: `gap-2`, `gap-4` (일관성 없음)
- AFTER: `gap-1.5`, `gap-3` (균일한 간격)

## UX Improvements

### 1. 정보 밀도
- BEFORE: 일정 카드가 너무 커서 2-3개만 표시
- AFTER: 컴팩트한 디자인으로 3개 표시 + 더보기

### 2. 시각적 계층
- BEFORE: 모든 요소가 border로 둘러싸여 평평함
- AFTER: 그림자, 색상, 크기로 명확한 계층

### 3. 인터랙션 피드백
- BEFORE: 단순 색상 변경
- AFTER: smooth transition + 그림자 + ring

### 4. 모바일 최적화
- BEFORE: 작은 화면에서 복잡함
- AFTER: 더 많은 여백, 명확한 터치 영역

## Summary

### 제거한 것
- ❌ 불필요한 border
- ❌ 이중 박스 구조
- ❌ 진한 배경색
- ❌ 항상 표시되는 "일정 없음"
- ❌ 텍스트 이모지

### 추가한 것
- ✅ Accent borders (border-l-4)
- ✅ Subtle shadows
- ✅ 도트 인디케이터
- ✅ 원형 날짜 배지
- ✅ SVG 아이콘
- ✅ Smooth transitions
- ✅ Ring highlights
- ✅ Empty state 일러스트

### 결과
**"촌스러운 네모"** → **"노멀하고 세련된"** 캘린더
