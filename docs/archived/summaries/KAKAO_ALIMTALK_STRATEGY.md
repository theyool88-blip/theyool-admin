## 채널 친구 활용

### 알림톡 vs 친구톡
- **알림톡**: 전화번호만 있으면 발송 가능 (채널 친구 불필요)
- **친구톡**: 채널 친구에게만 발송 가능

### 채널 친구 = 마케팅 동의
- 채널 친구 추가 = 마케팅 수신 동의로 간주
- 친구에게만 이벤트, 프로모션 발송 가능

### 친구 전환 전략
- 알림톡에 "채널 추가" 버튼 포함
- 상담 시 QR코드로 채널 추가 안내
- 수임 시 채널 추가 권유

---

## 한계점

1. **채널 친구 식별 불가**
   - 알림톡 수신자가 채널 친구 추가해도 누군지 연결 안됨
   - 채널 친구는 카카오 닉네임으로만 표시

2. **의뢰인 관리는 우리 시스템에서**
   - luseed에서 전화번호 기반 관리
   - 채널 친구 목록과 별개로 운영

## 설정 순서 (채널 승인 후)

### 1단계: 발신프로필 등록
- 카카오 비즈메시지 → 발신프로필 관리
- 인증된 채널 연결
- 검수 신청 (1-2일)

### 2단계: 템플릿 등록
- 위 템플릿 5개 등록
- 각각 검수 신청 (1-3일)
- 거절 시 수정 후 재신청

### 3단계: Solapi 연동
- Solapi 콘솔 → 카카오톡 → 발신프로필 연동
- Sender Key 입력
- 환경변수 추가:
  ```
  KAKAO_SENDER_KEY=카카오_센더키
  KAKAO_PFID=플러스친구_ID
  ```

### 4단계: 코드 수정
- lib/notifications/kakao-alimtalk.ts 수정
- 실제 템플릿 코드 연동
- 테스트 발송
